I'm quite surprised of how fast it was to start coding with Vue, I'll try soon to use [Single File Components](https://vuejs.org/v2/guide/single-file-components.html) because, I suppose I really need to follow a way to structure my web app similary to Angular.

> [separation of concerns is not equal to separation of file types](https://vuejs.org/v2/guide/single-file-components.html#What-About-Separation-of-Concerns)

This is a nice example for how best practices and design patterns should be followed.

This is how single file vue components are defined:
```
<template>
<!-- Here the HTML -->
</template>

<script>
<!-- Here the component vue syntax -->
</script>

<style scoped>
<!-- Here the scoped css component -->
</style>
```
